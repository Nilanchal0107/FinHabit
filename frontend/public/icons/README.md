# FinHabits — PWA Icons

Place the following icon files in this directory before deploying:

| Filename              | Size       | Usage                          |
|-----------------------|------------|--------------------------------|
| favicon.svg           | SVG        | Browser tab favicon            |
| apple-touch-icon.png  | 180×180    | iOS home screen icon           |
| icon-72x72.png        | 72×72      | PWA icon (small)               |
| icon-96x96.png        | 96×96      | PWA icon                       |
| icon-128x128.png      | 128×128    | PWA icon                       |
| icon-144x144.png      | 144×144    | PWA icon                       |
| icon-152x152.png      | 152×152    | iPad icon                      |
| icon-192x192.png      | 192×192    | PWA icon (required, maskable)  |
| icon-384x384.png      | 384×384    | PWA icon (large)               |
| icon-512x512.png      | 512×512    | PWA splash screen (maskable)   |
| screenshot-mobile.png | 390×844    | Play Store / App Store preview |

## Design Guidelines
- Use FinHabits brand colors: Primary Indigo `#4F46E5` background
- Maskable icons: keep logo within central 80% safe zone
- Generate all sizes from a single 512×512 source SVG
- Recommended tool: https://maskable.app/editor
